Known Commands:

These are the generic commands that can be called by the end-user in a large variety of formulations.

This solution is making use of the built-in browser functionality for speech recognition and it uses some heuristics to determine which of the commands was called and to retrieve the parameters.

Everything happens client-side! :) This can result in a http POST request to an endpoint associated with the chosen command.
